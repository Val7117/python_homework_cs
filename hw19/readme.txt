1. Запустить клиент (udp_client.py) и сервер (udp_dns_server.py)
2. Отправлять с клиента запросы доменов.
3. Чтобы добавить переопределить запись нужно отправить с клиента комманду вида: ADD domain.name:ipv4_address. Например: ADD my.domain:17.17.17.17
4. Все логи отображаются на сервере и записываются в файл logs.txt, который находится в текущей директории.
