В файле hw7.py можно указать диапазон номеров для UIDs, [uid_start, uid_end]. По умолчанию стоят uid_start = 1000 и uid_end = 10000.
То есть будут находиться пользователи, начиная с 1000 по 10000 UID.
Также по умолчанию всегда будет отображаться UID пользователя 'root'.
Если вам нужен любой другой диапазон, то присвойте соответствующие номера UID-ов переменным uid_start=x и uid_end=y. Где x и y - начало и конец желаемого диапазона. 
Результат выполнения hw7.py записывается в файл output.txt
